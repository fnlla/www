# Support

## How To Get Help

- Usage and setup questions: open a GitHub Discussion (if enabled) or issue
- Bugs: open a bug report using the issue template
- Security issues: follow `SECURITY.md` and report privately

## What To Include

- Repository name and version/tag
- PHP and Composer versions
- Operating system
- Clear reproduction steps
- Expected vs actual behavior
- Logs or stack traces (with secrets removed)

## Scope

Support is provided for currently supported release lines and documented
capabilities in each repository.
